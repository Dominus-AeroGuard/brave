## Brave Backend

## Description

Backend Aero Guard

## Installation

```bash
$ npm install
$ npm run migration
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Runing the app using Docker

```bash
$ npm run docker:up
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
